# Centre for Indian Knowledge Systems and Holistic Education (SVNIT Surat)

**भारतीय ज्ञान परंपरा एवं समग्र शिक्षा केन्द्र**

The Centre for Indian Knowledge Systems and Holistic Education is a multidisciplinary research and education hub located at SVNIT Surat. This application serves as the official digital platform for the Centre, providing comprehensive information about its objectives, academic programmes, and research initiatives.

## Key Features

- **Branded Loading Experience**: A custom loading sequence featuring the Centre's official titles in Sanskrit and English with the SVNIT logo.
- **Structured Content Layout**: A vertical-stack skeleton designed for optimal readability and academic presentation, fully responsive across desktop and mobile devices.
- **Dynamic Photograph Display**: A high-impact hero carousel with random-shuffle functionality for showcasing visual highlights.
- **Integrated Research and Academic Sections**: Detailed tabs covering Objectives, Vision, Mission, Ph.D. Programme areas, and course offerings.
- **Paginated Image Gallery**: A paginated slider gallery that supports extensive collections of photographs with structured navigation and pagination controls.
- **Smooth Navigation Persistence**: Sticky navigation controls that track user progress and automatically highlight relevant sections during scroll.

## Technology Stack

- **Frontend Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Programming Language**: TypeScript
